# The Hibrythian Saga — Site

Official lore site for The Hibrythian Saga.

## Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS v3
- React Router v6
- Cloudflare Pages + Workers

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
Deployed via Cloudflare Pages. Wrangler config in `wrangler.toml`.
