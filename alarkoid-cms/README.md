# Alarkoid CMS

> The official CMS Dashboard for **The Hibrythian Saga** & **The Naiseikai Universe**  
> Built by Alarkius Elvya Jay

---

## What is Alarkoid CMS?

A minimal fantasy-themed CMS editor dashboard designed for large-scale lore management. It connects to Notion as the content source, uses GitHub as the source of truth, and deploys via Cloudflare Pages.

## Features

- 🏰 Fantasy-themed dark/light UI (Hibrythian & Naiseikai themes)
- 📝 Rich text block editor with full formatting toolbar
- 🌐 Live site preview pane
- 📄 Draggable page/subpage tree
- 🔄 Notion → GitHub → Cloudflare deploy pipeline (in progress)
- 🔒 Single-author, no public editing

## Stack

- HTML/CSS/JS prototype (current)
- Next.js app (planned)
- Notion API + GitHub Actions sync (planned)
- Cloudflare Pages deployment (planned)
